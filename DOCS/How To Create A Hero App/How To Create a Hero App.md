## Detailed Step-by-step Approach:

### Prepare the prompt for AI:
1. Open and copy the prompt file:
   [Sample Prompt for iFrame Plugin](Sample%20Prompt%20for%20Hero%20App.txt).
2. Paste and edit the sections that are marked with `(EDIT ME)` and fill in the required information. (Usually, filling the 1st and 3rd sections is enough.)
### Implement the hero app:
3. Open an AI coding tool like:
   * Google AI Studio
   * Base44
   * Bolt.new
   * Lovable.ai
   * GitHub Copilot Chat
   * Or any other AI coding tool you prefer.
5. Paste the edited prompt text there and ask the AI to implement the code for you.
6. Review and test it, making any necessary adjustments.

### Deploy the hero app:
7. Deploy the hero app:
   * **Google AI Studio:** You can deploy directly from there.
   * **Bolt.new:** You can deploy directly from there.
   * **Lovable.ai:** You can deploy directly from there.
   * **GitHub Copilot Chat:** You can deploy to your own hosting or use a service like Vercel, Netlify, or Render.
8. Get the deployed URL.

### Use the hero app in okDoc.ai:
9. Go to the okDoc.ai app (download the app or use the web version via [web.okdoc.ai](https://web.okdoc.ai)).
10. Hit the **Profile** button (bottom-right corner) -> **Heroes** -> **Hero Store**.
11. Paste the deployed URL in the "Load Hero from URL" section and hit the **Load** button.
12. After a successful load, go back to the Home screen (using the button on the bottom).
13. If the AI is active (yellow or red), hit the bubble to deactivate the AI, and then tap it again to activate it. If the AI is green, just hit the bubble.
14. You should now be able to interact with the app using voice commands according to your defined tools. Just say, "Hey Jarvis, please open the [hero app name] plugin," or give commands corresponding to the tools you have defined.

> *Note: A Hero App is technically a web app integrated into okDoc.ai as a plugin, allowing you to interact with it using your voice through our voice assistant interface.*
